# ExtraHop-NDR-Report
Excited to complete a hands-on project on ExtraHop NDR, analyzing threats like SUNBURST, Log4Shell, Zerologon, DCSync, and data exfiltration. Gained practical skills in threat detection, packet analysis, MITRE ATT&amp;CK mapping, triage, and incident response. 
Excited to share my latest hands-on cybersecurity project on **ExtraHop Network Detection & Response (NDR)**, where I worked on analyzing advanced threats, suspicious network behavior, and real-time incident investigations in an enterprise-like environment.

ExtraHop NDR is a powerful cybersecurity platform that provides deep visibility into network traffic, detects malicious activities, and helps security teams investigate threats faster through machine learning, behavioral analytics, and packet-level analysis. During this project, I explored how organizations can strengthen their security posture by identifying threats across endpoints, servers, cloud workloads, and network environments.

As part of the project, I reviewed and analyzed multiple high-risk detections such as **SUNBURST Command & Control activity, Log4Shell exploitation attempts, Zerologon attack attempts, DCSync attacks, Kerberos Golden Ticket attacks, suspicious DNS communications, SMB brute-force behavior, and potential data exfiltration events**. I also examined attacker techniques like lateral movement, privilege escalation, credential abuse, reconnaissance, and persistence mechanisms.

One of the key learning areas was understanding how detections are mapped to the **MITRE ATT&CK Framework**, which helped in identifying attacker tactics and techniques across different stages of an attack lifecycle. I also worked on the **Triage and Investigation modules**, where alerts are prioritized based on severity, risk scores, affected assets, and threat intelligence context.

Additionally, I explored **Assets, Records, and Packets** features to investigate suspicious hosts, review communication flows, and inspect packet-level evidence for deeper forensic analysis. This provided practical exposure to how analysts perform root cause analysis and validate threats using network telemetry.

🔹 Incident Response Workflows

This experience significantly improved my understanding of modern cyber threats and enterprise defense strategies. It also strengthened my practical skills required for **SOC Analyst, Cybersecurity Analyst, and Threat Detection roles**.
